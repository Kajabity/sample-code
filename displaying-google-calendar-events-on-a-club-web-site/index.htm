Google Calendar Demo

This page extracts the relevant parts of the Thistle Society events page to display the events list in a <div/> below. Refer to the original article, Displaying Google Calendar Events on a Club Web Site for more explanations.

The JavaScript retrieves the events from Google Calendar indirectly using a call to events-feed.php. It then extracts the event details and formats them as you (should) see below - picking out the date details and grouping by month.

The raw text description of the event is processed using Wiked Text (or Wiki Text till I get time to finish the rename). See http://www.kajabity.com/jquery-wikitext/ for more details.

You can also view these events on Google Calendar (opens in a new window).

It's broken! This sample used the Google V2 API, apparently... and they've deprecated it. I've changed it to use the "/basic" instead of "/full" view as this still works, but it's not perfect. I'll work on improving this version and then updating to version 3 API as soon as I can.
